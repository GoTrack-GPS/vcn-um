# Home Page

## How to build at local machine
- clone the repository 

- install python

- Change to site directory
```
cd /path/to/site/files
```

- Show Python version
```
python --version
```

- Launch web server (Python 2.x)
```
python -m SimpleHTTPServer
```

- Launch web server (Python 3.x)
```
python -m http.server
```
